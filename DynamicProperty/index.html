Protocol Dynamic​Property 

@available(iOS 13.0, macOS 10.15, tvOS 13.0, watchOS 6.0, *) public protocol DynamicProperty

An interface for a stored variable that updates an external property of a view.

The view gives values to these properties prior to recomputing the view's View/body-swift.property.

update() 

mutating func update()

Updates the underlying value of the stored value.

SwiftUI calls this function before rending a view's View/body-swift.property to ensure the view has the most recent value.
