Protocol Menu​Style 

@available(iOS 14.0, macOS 11.0, *) @available(tvOS, unavailable) @available(watchOS, unavailable) public protocol MenuStyle

A type that applies standard interaction behavior and a custom appearance to all menus within a view hierarchy.

To configure the current menu style for a view hiearchy, use the View/menuStyle(_:) modifier.
