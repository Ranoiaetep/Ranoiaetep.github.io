Structure Fetch​Request 

@available(iOS 13.0, macOS 10.15, tvOS 13.0, watchOS 6.0, *) @propertyWrapper public struct FetchRequest<Result>: DynamicProperty where Result: NSFetchRequestResult

A property wrapper type that makes fetch requests and retrieves the results from a Core Data store.

The fetch request and its results use the managed object context provided by the environment value EnvironmentValues/managedObjectContext.
