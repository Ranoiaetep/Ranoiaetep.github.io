Protocol Button​Style 

@available(iOS 13.0, macOS 10.15, tvOS 13.0, watchOS 6.0, *) public protocol ButtonStyle

A type that applies standard interaction behavior and a custom appearance to all buttons within a view hierarchy.

To configure the current button style for a view hierarchy, use the View/buttonStyle(_:)-7qx1 modifier. Specify a style that conforms to ButtonStyle when creating a button that uses the standard button interaction behavior defined for each platform. To create a button with custom interaction behavior, use PrimitiveButtonStyle instead.
