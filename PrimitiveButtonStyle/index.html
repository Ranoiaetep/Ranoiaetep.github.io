Protocol Primitive​Button​Style 

@available(iOS 13.0, macOS 10.15, tvOS 13.0, watchOS 6.0, *) public protocol PrimitiveButtonStyle

A type that applies custom interaction behavior and a custom appearance to all buttons within a view hierarchy.

To configure the current button style for a view hierarchy, use the View/buttonStyle(_:)-66fbx modifier. Specify a style that conforms to PrimitiveButtonStyle to create a button with custom interaction behavior. To create a button with the standard button interaction behavior defined for each platform, use ButtonStyle instead.
