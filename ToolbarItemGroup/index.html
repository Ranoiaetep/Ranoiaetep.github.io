Structure Toolbar​Item​Group 

@available(iOS 14.0, macOS 11.0, tvOS 14.0, watchOS 7.0, *) public struct ToolbarItemGroup<Content>: ToolbarContent where Content: View

A model that represents a group of ToolbarItems which can be placed in the toolbar or navigation bar.
