Protocol File​Document 

@available(iOS 14.0, macOS 11.0, *) @available(tvOS, unavailable) @available(watchOS, unavailable) public protocol FileDocument

A document model definition used to serialize documents to and from file contents.

Conformance to FileDocument is expected to have value semantics and be thread-safe. Serialization and deserialization will be done on a background thread.
