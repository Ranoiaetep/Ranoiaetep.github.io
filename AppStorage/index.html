Structure App​Storage 

@available(iOS 14.0, macOS 11.0, tvOS 14.0, watchOS 7.0, *) @frozen @propertyWrapper public struct AppStorage<Value>: DynamicProperty

A property wrapper type that reflects a value from UserDefaults and invalidates a view on a change in value in that user default.

Relationships

Conforms To

DynamicProperty

An interface for a stored variable that updates an external property of a view.

public init(wrappedValue: Value, _ key: String, store: UserDefaults? = nil) where Value == Data

Creates a property that can read and write to a user default as data.

Avoid storing large data blobs in user defaults, such as image data, as it can negatively affect performance of your app. On tvOS, a NSUserDefaultsSizeLimitExceededNotification notification is posted if the total user default size reaches 512kB.

public init(wrappedValue: Value, _ key: String, store: UserDefaults? = nil) where Value: RawRepresentable, Value.RawValue == Int

Creates a property that can read and write to an integer user default, transforming that to RawRepresentable data type.

A common usage is with enumerations:

enum MyEnum: Int { case a case b case c } struct MyView: View { @AppStorage("MyEnumValue") private var value = MyEnum.a var body: some View { ... } }
