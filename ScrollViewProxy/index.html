Structure Scroll​View​Proxy 

@available(iOS 14.0, macOS 11.0, tvOS 14.0, watchOS 7.0, *) public struct ScrollViewProxy

A proxy value allowing the scrollable views within a view hierarchy to be scrolled programmatically.

Methods

scroll​To(_:​anchor:​) 

public func scrollTo<ID>(_ id: ID, anchor: UnitPoint? = nil) where ID: Hashable

Scans all scroll views contained by the proxy for the first with a child view with identifier id, and then scrolls to that view.

If anchor is nil the container of the identified view will be scrolled the minimum amount to make the identified view wholly visible.

If anchor is non-nil it defines the points in the identified view and the scroll view that will be aligned, e.g. .top aligns the top of the identified view to the top of the scroll view, .bottom aligns the bottom of the identified view to the bottom of the scroll view, and so on.
