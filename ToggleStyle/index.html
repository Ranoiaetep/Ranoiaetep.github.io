Protocol Toggle​Style 

@available(iOS 13.0, macOS 10.15, tvOS 13.0, watchOS 6.0, *) public protocol ToggleStyle

A type that specifies the appearance and interaction of all toggles within a view hierarchy.

To configure the current toggle style for a view hiearchy, use the View/toggleStyle(_:) modifier.
