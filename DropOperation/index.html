Enumeration Drop​Operation 

@available(iOS 13.4, macOS 10.15, *) @available(tvOS, unavailable) @available(watchOS, unavailable) public enum DropOperation

Operation types that determine how a drag and drop session resolves when the user drops a drag item.

Enumeration Cases

cancel 

case cancel

Cancel the drag operation and transfer no data.

forbidden 

case forbidden

The drop activity is not allowed at this time or location.

copy 

case copy

Copy the data to the modified view.

move 

case move

Move the data represented by the drag items instead of copying it.

var hashValue: Int

The hash value.

Hash values are not guaranteed to be equal across different executions of your program. Do not save hash values to use during a future execution.

hashValue is deprecated as a Hashable requirement. To conform to Hashable, implement the hash(into:) requirement instead.

hash(into:​) 

public func hash(into hasher: inout Hasher)

Hashes the essential components of this value by feeding them into the given hasher.

Implement this method to conform to the Hashable protocol. The components used for hashing must be the same as the components compared in your type's == operator implementation. Call hasher.combine(_:) with each of these components.

Never call finalize() on hasher. Doing so may become a compile-time error in the future.
