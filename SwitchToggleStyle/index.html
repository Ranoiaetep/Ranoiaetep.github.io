Structure Switch​Toggle​Style 

@available(iOS 13.0, macOS 10.15, watchOS 6.0, *) @available(tvOS, unavailable) public struct SwitchToggleStyle: ToggleStyle

A toggle style that displays a leading label and a trailing switch.

To apply this style to a toggle, or to a view that contains toggles, use the View/toggleStyle(_:) modifier.
